# simple-output

version: 0.1.0

Output messages to stdout/stderr

## Getting Started
Install the module with: `npm install simple-output`

## Examples

```javascript
var log = require('simple-output');

log.success('Successful hello world');
```

## Available methods

- success(msg)
- error(msg)
- info(msg)
- message(msg)
- warning(msg)
